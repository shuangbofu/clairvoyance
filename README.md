## 图表

~~图表绘制字段使用的时候目前不是拖拽形式~~



支持图表类型：
折线图、柱状、指标卡
不同数量值和维度的情况下没有做区分，没有做限制，会导致图表的配置错误，会报错

图表中只有简单的数据和图形，没有其他



支持选择维度（设置字段名等）、数值（选择聚合函数）、筛选器（精确筛选、条件筛选、表达式）、图内筛选器
暂时不支持自定义字段（计算字段、分组字段）

暂不支持对比和次轴
