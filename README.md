# 在线分析-数据可视化
 千里眼 **Clairvoyance**

## 关于

开发一两天，所以只实现了最简单基础（数据源表等信息基于部分手动录入），只保证创建仪表盘、绘制图表等主流程走通。

## 数据源

目前直接配置在工作表中，后续单独维护

## 工作表

支持类型：mysql

后续hive，view（和表，比如关联表、聚合表等），上传数据，

目前手动维护在表中，后续可以读取数据库表信息直接同步数据表

## 仪表盘

可以创建仪表盘，新增图表

目前固定大小位置，不可拖动

## 图表

图表绘制字段使用的时候目前不是拖拽形式

支持图表类型：
折线图、柱状、指标卡
不同数量值和维度的情况下没有做区分，没有做限制，会导致图表的配置错误，会报错
